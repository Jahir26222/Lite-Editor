LiteEdit Pro is a lightweight, web-based design tool inspired by Figma. It allows users to create, move, and resize shapes and text elements on a canvas with real-time property editing and auto-save functionality.


üöÄ Features
Create Elements: Add rectangles and text boxes to the canvas.

Drag & Resize: Move elements freely or resize them from any of the four corners.

Text Editing: Double-click any text element to type directly inside it.

Properties Panel: Fine-tune X/Y coordinates, Width/Height, Rotation, and Colors.

Layers Management: View and select elements easily through the sidebar layer list.

Auto-Save: Your work is automatically saved to Local Storage, so it stays there even after a page reload.

Keyboard Shortcuts: Select an element and press Delete or Backspace to remove it.

Export Options:

JSON: Save the raw data of your project.

HTML: Export your design as a standalone, ready-to-use webpage.
